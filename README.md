# RaidGroupRoleIcons
A small add-on that displays a role icons next to players names in the raid panel. This is especially useful for raid leaders and anyone who sorts players via the raid panel.

## Preview
This is an example of the add-on in a raid group. Role icons for healers, tanks and DPS have been added.
